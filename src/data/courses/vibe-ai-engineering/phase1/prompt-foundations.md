## 4. The Industrial Task
- Master the **CO-STAR** framework (Context, Objective, Style, Tone, Audience, Response).
- Define the **System Aura** (The persona and behavioral constraints).

## 5. Framework Spotlight: CO-STAR
- **Context**: Provide background on the task.
- **Objective**: Define the exact goal.
- **Style**: Specify the writing style.
- **Tone**: Set the attitude.
- **Audience**: Identify who the output is for.
- **Response**: Define the format (JSON, Markdown, etc.).
