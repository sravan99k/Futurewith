## Key Metrics for AI Startups

### Product Metrics
- **DAU/MAU:** Daily/Monthly Active Users
- **Activation Rate:** % signing up who use product
- **Retention:** % still using after X days
- **Churn:** % leaving per month

### AI-Specific Metrics
- **Model Accuracy:** Precision, recall, F1 score
- **Inference Time:** Response latency
- **Cost per Inference:** API/Compute cost
- **Hallucination Rate:** % incorrect outputs

### Business Metrics
- **CAC:** Customer Acquisition Cost
- **LTV:** Lifetime Value
- **MRR/ARR:** Monthly/Annual Recurring Revenue
- **Gross Margin:** Revenue - COGS

### Healthy Ratios
- LTV:CAC > 3x
- CAC payback < 12 months
- Gross margin > 70%
- Churn < 5% monthly

## AI Pricing Models

### Common Models
1. **Subscription:** Fixed monthly/yearly fee
2. **Usage-based:** Pay per token/API call
3. **Freemium:** Free tier + paid upgrades
4. **Per-seat:** Per user per month
5. **Enterprise:** Custom pricing

### Pricing Benchmarks
- Entry AI SaaS: $29-$99/month
- Mid-market: $99-$499/month
- Enterprise: $499+/month
- API: $0.01-$0.10 per 1K tokens

## Quick Formulas

### Unit Economics
```
LTV = ARPU × Gross Margin × (1 / Churn Rate)
CAC Payback = CAC / (ARPU × Gross Margin)
Gross Margin = (Revenue - COGS) / Revenue
```

### Growth Metrics
```
MoM Growth = (This Month - Last Month) / Last Month
Net Revenue Retention = Expansion - Churn
Efficient Growth = Growth / Burn Rate
```

### AI Costs
```
Cost per Query = (Model Cost + Compute) / Queries
Cost per Token = Model API Cost / 1M tokens
```
